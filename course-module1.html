ğŸ¯ Key Interview Questions - Module 1

â“ Common Interview Questions

  1. Q: What is Flutter and why use it?
    A: Flutter is a cross-platform framework that lets you build apps for multiple platforms with a single codebase. Benefits include hot reload, native performance, and beautiful UI.

  2. Q: What's the difference between StatefulWidget and StatelessWidget?
    A: StatelessWidget is immutable and doesn't change. StatefulWidget can change over time and rebuild when setState() is called.

  3. Q: Explain Future vs Stream in Dart.
    A: Future represents a single async value that will arrive later (like ordering pizza). Stream represents multiple values over time (like a radio broadcast).

  4. Q: What is the Widget Tree?
    A: A hierarchical structure where each widget can contain child widgets, forming a tree from root to leaves.

  5. Q: When would you use const vs final?
    A: const is for compile-time constants (value known before app runs). final is for runtime constants (value set once when app runs, then never changes).
