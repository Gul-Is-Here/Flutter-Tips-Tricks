Lesson 6.1 ¬∑ Widget Rebuild Strategy

const constructors, Keys, selective rebuilds, memoization
üèóÔ∏è

Explain Like You're 5

Imagine rebuilding a LEGO city every time one window changes color. Instead, label each building (Key) and only repaint that window. Flutter prefers surgical updates over bulldozing entire rows.

Const widgets

Const constructors short-circuit rebuilds because Flutter reuses the existing widget instance.

Keys

Use ValueKey for semantic identity, ObjectKey for models, UniqueKey for forcing rebuilds.

Split widgets

Extract leaf widgets so only the smallest subtree rebuilds when a value changes.

class ProductRow extends StatelessWidget { const ProductRow({super.key, required this.product}); final Product product; @override Widget build(BuildContext context) { return Row( children: [ Expanded(child: Text(product.name)), PriceTag(price: product.price), // Separate widget ], ); } } class PriceTag extends StatelessWidget { const PriceTag({super.key, required this.price}); final double price; @override Widget build(BuildContext context) { return Text( '\$${price.toStringAsFixed(2)}', style: const TextStyle(fontFeatures: [FontFeature.tabularFigures()]), ); } }

Image tips

Use cached_network_image, specify cacheWidth, and prefer SVG for icons.

Memory pressure

Listen to didHaveMemoryPressure to drop caches. Monitor DevTools Memory tab.

Background parsing

Use compute() or Isolate.run for JSON so UI thread stays smooth.

Remember: For photo feeds, decode thumbnails server-side to avoid decoding 12MP images on the device.

üéØ Interview Drill

1. How do const widgets improve performance?

Flutter reuses const widget instances instead of rebuilding, reducing diffing work.

2. When would you use RepaintBoundary?

Around complex visuals (maps, charts) so their repaint cost doesn't cascade to parents.

3. How can you detect memory leaks?

Use DevTools memory tab, take heap snapshots, ensure streams/controllers are disposed.

4. Explain shader compilation jank.

On first display, shaders compile on-device causing a hitch. Warm them up and bundle the SKSL.

5. Difference between debug and profile mode?

Debug enables asserts/hot reload but isn't optimized; profile runs near-release performance with instrumentation.
