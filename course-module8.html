Lesson 8.1 ¬∑ Build Modes & Flavors

Debug vs profile vs release, build flavors, environment configs
üç±

Explain Like You're 5

Think of shipping lunch boxes. Debug mode is a sample bite for chefs, profile mode is a tasting portion for critics, and release mode is the sealed lunch sold to customers.

Build modes

debug (hot reload, asserts), profile (performance), release (optimized, sized down).

Flavors

Configure different bundle IDs, icons, Firebase configs (e.g., staging vs production).

Env configs

Use flutter_dotenv or generated Dart files to inject API base URLs per flavor.

# android/app/build.gradle android { flavorDimensions "default" productFlavors { dev { applicationId "com.gul.fluttertips.dev" resValue "string", "app_name", "Tips (Dev)" } prod { applicationId "com.gul.fluttertips" resValue "string", "app_name", "Flutter Tips" } } } # ios/Runner/Configs/Debug-dev.xcconfig PRODUCT_BUNDLE_IDENTIFIER = com.gul.fluttertips.dev
1. Create flavor configs (gradle, xcconfig)
2. Wire environment constants (dart defines)
3. Update CI to build each flavor

Requirement Google Play App Store
Binary format AAB (App Bundle) IPA via Xcode archive
Signing Upload key + Play App Signing Distribution certificate + provisioning profile
Assets Screenshots per density, feature graphic 6.7/6.5/5.5-inch screenshots, app preview video optional

üéØ Interview Drill

1. Difference between profile and release mode?

Profile keeps some instrumentation for performance tracing, release strips it all for final binaries.

2. How do you manage multiple environments?

Use build flavors/dart defines to change API hosts, analytics keys, and icons.

3. Required Android assets?

Adaptive icons, feature graphic, up to 8 screenshots per form factor, data safety declaration.

4. How to automate releases?

CI workflow triggered on tags, builds Android/iOS artifacts, signs them, uploads to Play/App Store via Fastlane or APIs.

5. Monitoring after release?

Crashlytics/Sentry for crashes, Firebase Performance for latency, analytics for retention.
